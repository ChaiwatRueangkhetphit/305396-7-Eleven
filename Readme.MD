# 7-Eleven

1. แสดง Output คำว่า Seven เมื่อตัวเลข Input ที่ให้ไปหารด้วย 7 ลงตัว

3. แสดง Output คำว่า Seven-Eleven เมื่อตัวเลข Input ที่ให้ไปหารด้วย 7 และ 11 ลงตัว
